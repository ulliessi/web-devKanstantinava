Opening Developer’s Console in Browsers:

  • Mozilla Firefox: You can open the developer console in Firefox by pressing Ctrl + Shift + I or F12 on Windows and Linux, or Cmd + Opt + I on macOS.
  • Safari: To open the developer console in Safari, first, you need to enable the Develop menu. Go to Safari > Preferences > Advanced and check the box for “Show Develop Menu in Menu Bar”. Then, you can open the console by selecting Show JavaScript Console from the Develop menu or by using the keyboard shortcut

Question 2

Absolute vs Relative Links:

  • Absolute Links: These are links that contain the complete URL, including the protocol and domain name. They are independent of the current page’s location and can be used to access resources from any location. Absolute links are generally preferable when linking to resources on different websites.
  • Relative Links: These are links that specify the location of a resource relative to the current page’s location. They are most practical when used within the same website, as they are shorter and often more portable. Relative links are generally preferable when linking to resources within the same website.

Question 3

Image Formats in Web Development:

  • JPG: This format is a good choice for lossy compression of still images. It’s the most popular for this purpose. However, it should be avoided when more precise reproduction of the image is required.
  • GIF: This format is most practical for simple images and animations. However, it has limited colors (maximum of 256), making it a poor choice for displaying most photography.
  • PNG: This format is preferred over JPEG for more precise reproduction of source images, or when transparency is needed. However, newer formats like WebP and AVIF provide even better compression and reproduction.
  • WEBP: This format offers much better compression than PNG or JPEG with support for higher color depths, animated frames, transparency, etc. However, it is not quite as well-supported in browsers and does not support progressive rendering.
  • SVG: This is a vector image format, ideal for user interface elements, icons, diagrams, etc., that must be drawn accurately at different sizes.
